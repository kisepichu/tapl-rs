# tapl-rs

A Rust implementation of exercises from *Types and Programming Languages* (Japanese edition).


## Untyped lambda calculus($\lambda$)

[`untyped`](https://github.com/kisepichu/tapl-rs/tree/main/untyped)

```
$ cargo run --bin untyped
> (\\1 0)\0
input: (\\1 0) \0
   ->* \(\0) 0
> 
CTRL-C

$ 
```
